## 📂 精选项目

### 📖 [KirinWiki](https://github.com/BoooSAMA/KirinWiki)

个人维基/博客网站 — 使用 Astro 构建，记录学习历程、分享技术见解，并探索现代 Web 技术。

`Astro` `TypeScript` `Tailwind CSS` `Preact` `Cloudflare Workers`

### 🍞 [Smart Bakery](https://github.com/BoooSAMA/Smart-Bakery)

Flutter 移动应用，用于智能烘焙房实时监控 — 通过局域网连接 Raspberry Pi 后端，显示温湿度读数并控制风扇、蜂鸣器和静音模式。

`Dart` `Flutter` `IoT` `Raspberry Pi`

### 📝 [Online Examination](https://github.com/BoooSAMA/Online-Examination)

在线考试平台 — 包含管理员仪表盘、自动评分、模拟测试、限时考试和跨多个教育级别的自动生成数学题。

`C#` `ASP.NET Core` `Blazor` `EF Core` `SQL Server`

### 📺 [BiliLive](https://github.com/BoooSAMA/dart_simple_live_bilibili)

专注哔哩哔哩直播的 Android 客户端 — 基于 [Simple Live](https://github.com/xiaoyaocz/dart_simple_live) 精简优化，支持分区浏览、弹幕显示、多清晰度播放和本地收藏/关注（无需登录）。

`Dart` `Flutter` `哔哩哔哩` `Android`
